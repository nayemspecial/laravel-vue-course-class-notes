**প্লাটফর্ম:** প্রযুক্তিপ্লাস একাডেমী (ProjuktiPlus Academy)

**বিষয়:** ১০.৫ ট্যাগ এডিট করার জন্য এডিট ফর্ম সেটআপ করা, সাকসেস নোটিফিকেশন মেসেজ শো করানো, এডিট ও আপডেট রাউট যোগ করা, ট্যাগ কন্ট্রোলারে এডিট মেথড তৈরি করা, ট্যাগ লিস্ট পেজে এডিট লিংক যুক্ত করা, এডিট ফর্ম ডাইনামিক করা। 

**প্রযুক্তি প্লাস একাডেমির** পক্ষ থেকে আপনাদের স্বাগতম। গত পর্বে আমরা লারাভেলে কীভাবে ডাটাবেজে ডাটা স্টোর করতে হয় তা দেখেছি। আজকের পর্বে আমরা কীভাবে লারাভেলে ট্যাগ এডিট করার জন্য এডিট ফর্ম সেটআপ করতে হয়, রুট মডেল বাইন্ডিং (Route Model Binding) ব্যবহার করতে হয় এবং এডিট ফর্মটি ডাইনামিক করতে হয়—তার বিস্তারিত প্রক্রিয়া শিখব।

নিচে ধাপভিত্তিক সম্পূর্ণ কোড এবং বিস্তারিত ব্যাখ্যা দেওয়া হলো:

## ১. সাকসেস নোটিফিকেশন মেসেজ শো করানো

ট্যাগ ক্রিয়েট বা আপডেট হওয়ার পর ব্যবহারকারীকে সাকসেস মেসেজ দেখানোর জন্য `index.blade.php` ফাইলের শুরুতে নিচের কোডটি যুক্ত করুন:

```html
@if (session('success'))
    <div class="bg-green-100 border border-green-400 text-green-700 px-4 py-3 rounded mb-4">
        {{ session('success') }}
    </div>
@endif

```

## ২. এডিট ও আপডেট রাউট যোগ করা (`routes/web.php`)

ট্যাগ এডিট ফর্ম প্রদর্শন এবং ডাটা আপডেট করার জন্য `web.php` ফাইলে রাউট যোগ করুন:

```php
use App\Http\Controllers\Admin\TagController;

// এডিট ফর্ম দেখানোর জন্য গেট (GET) রাউট
Route::get('/admin/tags/{tag}/edit', [TagController::class, 'edit'])->name('admin.tags.edit');

// ডাটা আপডেট করার জন্য পুট (PUT) রাউট
Route::put('/admin/tags/{tag}', [TagController::class, 'update'])->name('admin.tags.update');

```

## ৩. কন্ট্রোলারে এডিট মেথড তৈরি করা (`TagController.php`)

লারাভেলের **Route Model Binding** ব্যবহার করে সহজেই নির্দিষ্ট ট্যাগের তথ্য রিটার্ন ভিউতে পাঠানো যায়:

```php
// এডিট ফর্ম ভিউ রিডাইরেক্ট করা
public function edit(Tag $tag)
{
    return view('admin.tags.edit', compact('tag'));
}

```

## ৪. ট্যাগ লিস্ট পেজে এডিট লিংক যুক্ত করা (`index.blade.php`)

ট্যাগ টেবিলের অ্যাকশন কলামে থাকা এডিট বাটনে নির্দিষ্ট ট্যাগের আইডি বা অবজেক্ট পাস করে লিংক তৈরি করুন:

```html
<a href="{{ route('admin.tags.edit', $tag) }}" class="btn-secondary">
    এডিট
</a>

```

## ৫. এডিট ফর্ম ডাইনামিক করা (`edit.blade.php`)

`resources/views/admin/tags/edit.blade.php` ফাইলে ফর্মের অ্যাকশন, মেথড এবং ইনপুট ভ্যালু ডাইনামিক সেটআপ করুন:

```html
@extends('admin.layouts.app')

@section('content')
<div class="max-w-2xl mx-auto bg-white p-6 rounded-lg shadow">
    <h2 class="text-xl font-bold mb-4">ট্যাগ এডিট করুন</h2>

    <!-- এডিট ফর্ম -->
    <form action="{{ route('admin.tags.update', $tag) }}" method="POST">
        @csrf
        @method('PUT')

        <!-- ট্যাগ নেম ইনপুট -->
        <div class="mb-4">
            <label for="name" class="block text-gray-700 font-medium mb-2">ট্যাগের নাম</label>
            <input type="text" name="name" id="name" 
                   value="{{ old('name', $tag->name) }}" 
                   class="w-full border-gray-300 rounded-md shadow-sm @error('name') border-red-500 @enderror">
            
            @error('name')
                <p class="text-red-500 text-xs mt-1">{{ $message }}</p>
            @enderror
        </div>

        <!-- বর্তমান স্লাগ প্রদর্শন -->
        <div class="mb-4 text-sm text-gray-500">
            বর্তমান স্লাগ: <code class="bg-gray-100 px-2 py-1 rounded">{{ $tag->slug }}</code>
        </div>

        <!-- সাবমিট ও বাতিল বাটন -->
        <div class="flex items-center gap-3">
            <button type="submit" class="bg-blue-600 text-white px-4 py-2 rounded-md hover:bg-blue-700">
                আপডেট করুন
            </button>
            <a href="{{ route('admin.tags.index') }}" class="bg-gray-500 text-white px-4 py-2 rounded-md hover:bg-gray-600">
                ফিরে যান
            </a>
        </div>
    </form>
</div>
@endsection

```

## ৬. কোডের লজিক্যাল ব্যাখ্যা

* **Route Model Binding:** রাউটে `{tag}` ব্যবহার করায় এবং কন্ট্রোলারের টাইপ-হিন্ট `Tag $tag` রাখায় লারাভেল স্বয়ংক্রিয়ভাবে ডাটাবেজ থেকে আইডি অনুযায়ী নির্দিষ্ট ট্যাগটি ফিল্টার করে নিয়ে আসে। যদি ট্যাগ না পাওয়া যায়, তবে এটি স্বয়ংক্রিয়ভাবে `404 Not Found` রিটার্ন করবে।


* **`@method('PUT')`:** এইচটিএমএল ফর্ম সাধারণত `GET` এবং `POST` সাপোর্ট করে। লারাভেলে আপডেট রিকোয়েস্ট পাঠানোর জন্য ফর্মে `POST` মেথড রেখে ভেতরে `@method('PUT')` ব্লেড ডিরেক্টিভ ব্যবহার করতে হয়।


* **`old('name', $tag->name)`:** যদি ভ্যালিডেশন ভুল হয়, তবে ব্যবহারকারীর টাইপ করা নতুন ইনপুট (`old('name')`) ধরে রাখবে; অন্যথায় ডাটাবেজের আগের নাম (`$tag->name`) দেখাবে।
