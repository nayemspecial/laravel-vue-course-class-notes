# ক্লাস নোট: লারাভেল ফ্যাক্টরি এবং সিডার (Laravel Factory & Seeder)

**প্রজেক্ট:** প্রজু্ক্তি প্লাস একাডেমী (ProjuktiPlus Academy)

**বিষয়:** ৮.২ ইউজার ফ্যাক্টরি কাস্টমাইজেশন, গুরুত্বপূর্ণ ক্লাস ও প্রপার্টির কাজ, ডেফিনেশন মেথড, ফেকার লাইব্রেরিতে বাংলাদেশি ডাটা (bn_BD) সেটআপ করা।

---

## ১. প্রজেক্ট ডিরেক্টরি এবং প্রাথমিক সেটআপ

* ডাটাবেজ নিয়ে কাজ করার সময় প্রধানত দুটি ফোল্ডার নিয়ে কাজ করতে হয়:
* `database/factories/`
* `database/seeders/`


* ডিফল্টভাবে লারাভেলে `UserFactory.php` ফাইলটি দেওয়া থাকে। আমরা এই ফাইলটি এডিট করে আমাদের কাস্টম ডামি ডাটা জেনারেট করব।
* কোড এডিটর বা টার্মিনালে কাজ করার সুবিধার্থে আলাদা গিট ব্যাশ (Git Bash) টার্মিনাল ওপেন করে নেওয়া ভালো।

---

## ২. ইউজার ফ্যাক্টরি ফাইলের গঠন বিশ্লেষণ

`UserFactory.php` ফাইলের উপরের অংশে কিছু গুরুত্বপূর্ণ নেমস্পেস এবং ক্লাস ইম্পোর্ট করা থাকে:

```php
namespace Database\Factories;

use App\Models\User;
use Illuminate\Database\Eloquent\Factories\Factory;
use Illuminate\Support\Facades\Hash;
use Illuminate\Support\Str;

```

### গুরুত্বপূর্ণ ক্লাস ও প্রপার্টির কাজ:

* **`User` মডেল:** ফ্যাক্টরি ক্লাসটি কার জন্য ডামি ডাটা তৈরি করবে তা ডিফাইন করে।
* **`Hash` ক্লাস:** পাসওয়ার্ড হ্যাশ করতে যে কর্মকাণ্ডগুলো করা লাগে তা এর মধ্যে বলা আছে।
* **`Str` ক্লাস:** স্ট্রিং হেলপার হিসেবে কাজ করে (যেমন: রেন্ডম স্ট্রিং তৈরি করা)।
* **স্ট্যাটিক পাসওয়ার্ড প্রপার্টি (`protected static ?string $password;`):** সব ইউজারের ক্ষেত্রে একই হ্যাশ পাসওয়ার্ড বারবার পুনরায় ব্যবহার করে পারফরমেন্স অপটিমাইজেশন নিশ্চিত করার জন্য এটি ব্যবহার করা হয়।

## ৩. ডেফিনেশন মেথড (`definition()`) এবং কাস্টম কোডিং

লারাভেল ফ্যাক্টরিতে ডামি ডাটার স্ট্রাকচার তৈরি করতে হয় `definition()` মেথডের ভেতর, যার রিটার্ন টাইপ হচ্ছে একটি অ্যারে (`array`)।

### ক) বাংলা ডামি ডাটা সেটআপ (Faker Localization)

আমাদের প্রজেক্টে অনেক বাংলা ওয়ার্ডের প্রয়োজন হওয়ায় আমরা ফেকার (Faker) লাইব্রেরিতে বাংলাদেশি লোকেল ব্যবহার করব:

```php
$faker = \Faker\Factory::create('bn_BD');

```

### খ) ইউনিক ও সেফ ইমেইল জেনারেট করা

ইমেইল যেন সবসময় ইউনিক এবং ছোট হাতের (lower case) অক্ষরে তৈরি হয়, তা এনসিওর করতে নিচের লজিকটি ব্যবহার করা হয়েছে:

```php
$safeEmail = Str::lower(Str::random(8)) . '.' . $faker->unique()->safeEmail;

```

### গ) কাস্টম ফিল্ডসমূহের লজিক বিশ্লেষণ

আমাদের ইউজার টেবিলে ডিফল্ট ফিল্ডের পাশাপাশি কিছু কাস্টম ফিল্ড আছে। সেগুলোর ডামি ডাটা জেনারেশন মেথডসমূহ নিচে দেওয়া হলো:

* **`avatar`:** প্রাভাটার (`pravatar.cc`) থেকে ১৫০ সাইজের ইমেজ ইউআরএল নিব এবং ১ থেকে ৭০ এর মধ্যে রেন্ডম নাম্বার কনক্যাটিনেট করব।
* **`bio`:** রিয়েল টেক্সট (`realText()`) মেথড দিয়ে ১০০ ক্যারেক্টারের আসল শব্দের টেক্সট তৈরি করব (লরেম ইপসাম নয়)।
* **`expertise`:** `randomElement()` মেথডের মাধ্যমে নির্দিষ্ট কিছু বাংলা ও ইংরেজি বিষয়ের অ্যারে থেকে রেন্ডমলি একটি এলিমেন্ট সিলেক্ট করব।
* **`role`:** ব্যবহারকারীদের বিভিন্ন রোল (जैसे: reader, author, admin, super_admin) রেন্ডমলি অ্যাসাইন করব।
* **`is_featured`:** বুলিয়ান মেথডে ২০% সম্ভাবনা (probability) সেট করে দিব, যাতে ১০০ জনের মধ্যে ২০ জন ফিচারড এবং ৮০ জন নন-ফিচারড লেখক হিসেবে সেটআপ হয়।

## ৪. সম্পূর্ণ কোড স্ট্রাকচার (`UserFactory.php`)

নিচে আমাদের কাস্টমাইজড `UserFactory` এর সম্পূর্ণ কোড দেওয়া হলো:

```php
public function definition(): array
{
    // বাংলা ডামি ডাটার জন্য ফেকার ফ্যাক্টরি ক্রিয়েট
    $faker = \Faker\Factory::create('bn_BD');

    // ছোট হাতের অক্ষরে ইউনিক ও সেফ ইমেইল তৈরি
    $safeEmail = Str::lower(Str::random(8)) . '.' . $faker->unique()->safeEmail;

    return [
        'name' => fake()->name(),
        'email' => $safeEmail,
        'email_verified_at' => now(),
        'password' => static::$password ??= Hash::make('password'),
        'remember_token' => Str::random(10),
        'avatar' => 'https://pravatar.cc/150?img=' . $faker->numberBetween(1, 70),
        'bio' => $faker->realText(100),
        'expertise' => $faker->randomElement([
            'এআই এক্সপার্ট', 
            'প্রোগ্রামিং বিশেষজ্ঞ', 
            'ইউআই ইউএক্স ডিজাইনার', 
            'ডেটা সায়েন্টিস্ট', 
            'ক্লাউড আর্কিটেক্ট', 
            'সাইবার নিরাপত্তা', 
            'মোবাইল অ্যাপ ডেভেলপার', 
            'ওয়েব ডেভেলপার'
        ]),
        'role' => $faker->randomElement(['reader', 'author', 'admin', 'super_admin']),
        'is_featured' => $faker->boolean(20), // ২০% ট্রু হওয়ার সম্ভাবনা
    ];
}

```
