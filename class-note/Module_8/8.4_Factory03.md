# ক্লাস নোট: লারাভেল ফ্যাক্টরি এবং সিডার (Laravel Factory & Seeder)

**প্লাটফর্ম:** প্রযুক্তিপ্লাস একাডেমী

**বিষয়:** ৮.৪ ক্যাটাগরি ফ্যাক্টরি তৈরি, নেমিং কনভেনশন এক্সপ্লেইন, ইউনিক নেম ও ইউআরএল ফ্রেন্ডলি স্লাগ জেনারেট করা।


---

### ১. নতুন ফ্যাক্টরি তৈরি করা (Creating the Factory)

প্রথমে আমাদের যে টার্মিনাল আছে সেই টার্মিনালটাকে ওপেন করব। ওপেন করে আমরা ক্যাটাগরি মডেলের জন্য একটি ফ্যাক্টরি তৈরি করতে নিচের কমান্ডটি লিখতে পারি:

```bash
php artisan make:factory CategoryFactory --model=Category

```

**নেমিং কনভেনশন নোট:**
মডেল সবার আগে ক্রিয়েট করি বলে মডেলে আমরা জাস্ট শব্দটা লিখে দিই (যেমন: `Category`, `User`, `Post`)। কিন্তু এরপরে আপনি যত ক্লাস ক্রিয়েট করবেন, সব ক্লাসের সাথে তার পরিচয় বা বৈশিষ্ট্যটা যেন বলে দেন (যেমন: `CategoryFactory`)। লারাভেল আমাদেরকে এই কনভেনশনটাই মেইনটেইন করতে বলে। কমান্ডে `--model=Category` বলে দেওয়ার কারণে এই ফ্যাক্টরিটা আমাদের ক্যাটাগরি মডেলের সাথে এটাচ বা ফিক্স হয়ে যাবে।

### ২. ফ্যাক্টরি লোকালাইজেশন (Localization Setup)

ফ্যাক্টরি তৈরি হওয়ার পর আমরা আমাদের ডিরেক্টরি থেকে `CategoryFactory` ফাইলটি ওপেন করব। লারাভেলে ডিফল্ট ক্লাসের ডেফিনেশন সুন্দর করে দেওয়া থাকে। আমরা ডেটাগুলোকে বাংলা ভাষায় জেনারেট করার জন্য `definition()` মেথডের শুরুতে ফ্যাক্টরি লোকালাইজেশন সেটআপ করতে পারি:

```php
$faker = \Faker\Factory::create('bn_BD'); // বাংলা ভাষা ফর বাংলাদেশ

```

*নোট: পূর্বে আমরা এই কোডটি ভুলবশত রিটার্নের মধ্যে করার কথা ভাবলেও, এটি মূলত রিটার্নের উপরে `definition()` মেথডের বডিতে লিখতে হবে। রিটার্নের মধ্যে আমরা শুধু অ্যাসোসিয়েটিভ অ্যারের ফিল্ডের নামগুলো রাখব।*

### ৩. ইউনিক নেম ও ইউআরএল ফ্রেন্ডলি স্লাগ জেনারেট (Unique Name & Slug Setup)

ক্যাটাগরি নেম যেন ডুপ্লিকেট না হয় এবং স্লাগ যেন ইউআরএল ফ্রেন্ডলি হয়, সেজন্য আমরা রিটার্নের উপরে ভেরিয়েবল নিয়ে কোডটি গুছিয়ে নেব। স্লাগের জন্য আমাদের স্ট্রিং হেল্পার দরকার হবে, যা আমরা `Illuminate\Support\Str` ডিরেক্টরি থেকে পাবো।

```php
// ইউনিক নামের ফরমেটিং
$name = $this->faker->unique()->words(2, true); 
// এখানে ২ দিয়ে বুঝাচ্ছি দুটি শব্দ আসবে এবং ট্রু (true) দিয়ে স্ট্রিং হিসেবে রিটার্ন করবে। 
// ইউনিক বলার কারণে ডুপ্লিকেট নেম রিমুভ হবে।

// ইউআরএল ফ্রেন্ডলি স্লাগ তৈরি
$slug = \Illuminate\Support\Str::slug($name) . '-' . \Illuminate\Support\Str::random(4);
// নামের স্পেসগুলো সরিয়ে হাইফেন দিয়ে প্রতিস্থাপিত করবে এবং শেষে ৪ অক্ষরের একটি রেন্ডম স্ট্রিং যুক্ত করবে।

```

### ৪. সম্পূর্ণ ক্যাটাগরি ফ্যাক্টরি কোড স্ট্রাকচার (Complete Factory Code)

আমাদের ক্যাটাগরি টেবিলের বাকি ফিল্ডগুলো (যেমন: `description`, `icon`, `color`, `is_active`, `sort_order`) সহ সম্পূর্ণ কোডটি নিচে সাজিয়ে দেওয়া হলো। আইকন ও কালার থিমের জন্য আমরা উপরে আলাদা অ্যারে ডিফাইন করে `randomElement` ব্যবহার করেছি যেন কোড দেখতে ভালো লাগে:

```php
<?php

namespace Database\Factories;

use App\Models\Category;
use Illuminate\Database\Eloquent\Factories\Factory;
use Illuminate\Support\Str;

class CategoryFactory extends Factory
{
    /**
     * The name of the factory's corresponding model.
     *
     * @var string
     */
    protected $model = Category::class;

    /**
     * Define the model's default state.
     *
     * @return array<string, mixed>
     */
    public function definition(): array
    {
        // বাংলা ভাষা ফর বাংলাদেশ সেটআপ
        $faker = \Faker\Factory::create('bn_BD');

        // ওয়েবসাইট বা থিমে ব্যবহৃত আইকনগুলোর অ্যারে
        $icons = [
            'fa-laptop', 'fa-book', 'fa-mobile', 'fa-gamepad', 'fa-camera',
            'fa-heart', 'fa-car', 'fa-music', 'fa-plane', 'fa-shirt'
        ];

        // থিমের মেইন হেক্সা কালারগুলোর অ্যারে
        $colors = ['#FF5733', '#33FF57', '#3357FF', '#F3FF33', '#FF33F3', '#33FFF0'];

        // নাম ও স্লাগ লজিক
        $name = $this->faker->unique()->words(2, true);
        $slug = Str::slug($name) . '-' . Str::random(4);

        return [
            'name'        => $name,
            'slug'        => $slug,
            'description' => $this->faker->sentence(15), // ১৫টি শব্দের একটি সেন্টেন্স
            'icon'        => $this->faker->randomElement($icons), // অ্যারে থেকে রেন্ডম আইকন নেবে
            'color'       => $this->faker->randomElement($colors), // অ্যারে থেকে রেন্ডম কালার নেবে
            'is_active'   => true, // ক্যাটাগরিগুলো সরাসরি একটিভ বলে দিচ্ছি
            'sort_order'  => $this->faker->numberBetween(0, 10), // ০ থেকে ১০ এর মধ্যে রেন্ডম অর্ডার থাকবে
        ];
    }
}

```

### সারসংক্ষেপ:

আমরা ক্যাটাগরি টেবিলের কালার, আইকন, ডেসক্রিপশন এবং রেন্ডম অর্ডারের জন্য `numberBetween` সহ সব রিকোয়ারমেন্ট সাকসেসফুলি সেটআপ করেছি। এই ডেটাগুলোর উপর ভিত্তি করে এখন আমরা খুব সহজেই ডেটাবেজ সিডিং সম্পন্ন করতে পারব। আজকের ক্লাস এখানেই শেষ করছি। সবাইকে ধন্যবাদ!
