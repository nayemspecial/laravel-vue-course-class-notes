# ক্লাস নোট: লারাভেল ফ্যাক্টরি এবং সিডার (Laravel Factory & Seeder)

**প্রজেক্ট:** প্রজু্ক্তিপ্লাস একাডেমী (ProjuktiPlus Academy)

**বিষয়:** ৮.৫ ট্যাগ ফ্যাক্টরি তৈরি, ফিল্ড ও ভেরিয়েবল ডিফাইন করা, ইউআরএল ফ্রেন্ডলি স্লাগ তৈরি। 

প্রযুক্তিপ্লাস একাডেমির পক্ষ থেকে আপনাদের সবাইকে স্বাগতম জানাচ্ছি। গত ক্লাসে আমরা ক্যাটাগরি ফ্যাক্টরি ক্রিয়েট করে তার ভেতরে প্রয়োজনীয় কোড লিখেছিলাম। আজকে আমরা পরবর্তী ফ্যাক্টরি অর্থাৎ **ট্যাগ ফ্যাক্টরি (Tag Factory)** তৈরি করব এবং তার কোড সেটআপ করব।

নিচে ক্লাসের আলোচনার ওপর ভিত্তি করে সম্পূর্ণ নোট ও কোড স্ট্রাকচার দেওয়া হলো:

---

### ১. নতুন ট্যাগ ফ্যাক্টরি তৈরি করা (Creating the Tag Factory)

ফ্যাক্টরির নেমগুলো সাধারণত এমন হয় যে নামের শেষে ফ্যাক্টরি শব্দটা থাকে (যেমন: `TagFactory`, `CommentFactory`)। ফ্যাক্টরি ক্রিয়েট করার সময় আমরা মডেলকে এটার সাথে এলাইন বা কানেক্ট করে দিতে পারি।

টার্মিনালে আপ অ্যারো (Up Arrow) ইউজ করে গত কমান্ডটি এনে বা নতুন করে ড্যাশ ড্যাশ মডেল (`--model`) অপশন যুক্ত করে আমরা নিচের কমান্ডটি রান করব:

```bash
php artisan make:factory TagFactory --model=Tag

```

*নোট: পিএইচপি আরটিজান মেক ফ্যাক্টরি ট্যাগ ফ্যাক্টরি লেখার পর `--model=Tag` ব্যবহার করলে লারাভেল আমাদের ট্যাগ মডেলের সাথে এই ট্যাগ ফ্যাক্টরিটাকে এলাইন করে দেয়।*

---

### ২. ফিল্ড ও ভেরিয়েবল ডিফাইন করা (Defining Fields & Variables)

আমাদের ট্যাগ ফ্যাক্টরিতে মূলত জাস্ট দুইটা ফিল্ড থাকবে: `name` এবং `slug`। এই ফিল্ডগুলোর ডেফিনেশন বা ফরম্যাট ঠিক করার জন্য আমরা অ্যাসোসিয়েটিভ অ্যারের `return` ব্লকের ঠিক আগে ভেরিয়েবলগুলো ডিফাইন করে নেব।

বাংলাদেশি লোকাল ডেটা জেনারেট করার জন্য আমাদের এখানে একটা অবজেক্ট ইনস্ট্যান্স করতে হয়, যেটাকে আমরা ফেকার ইনস্ট্যান্স বলি। আমরা আমাদের ক্যাটাগরি ফ্যাক্টরি থেকে এই ইনস্ট্যান্স ও ফরম্যাটের কোডগুলো নিয়ে এখানে ব্যবহার করতে পারি।

---

### ৩. সম্পূর্ণ ট্যাগ ফ্যাক্টরি কোড স্ট্রাকচার (Complete Tag Factory Code)

আমাদের ক্যাটাগরি ফ্যাক্টরির লজিক ও স্ট্রাকচারের সাথে মিল রেখে ট্যাগ ফ্যাক্টরির সম্পূর্ণ কোড নিচে দেওয়া হলো:

```php
<?php

namespace Database\Factories;

use App\Models\Tag;
use Illuminate\Database\Eloquent\Factories\Factory;
use Illuminate\Support\Str;

class TagFactory extends Factory
{
    /**
     * The name of the factory's corresponding model.
     *
     * @var string
     */
    protected $model = Tag::class;

    /**
     * Define the model's default state.
     *
     * @return array<string, mixed>
     */
    public function definition(): array
    {
        // ফেকার ইনস্ট্যান্স এবং বাংলাদেশি ডাটা জেনারেট সেটআপ
        $faker = \Faker\Factory::create('bn_BD');

        // নেম জেনারেশন লজিক (ইউনিক এবং দুটি শব্দ বিশিষ্ট স্ট্রিং)
        $name = $this->faker->unique()->words(2, true);

        // স্লাগ জেনারেশন লজিক (ইউআরএল ফ্রেন্ডলি ও ইউনিক করার জন্য রেন্ডম স্ট্রিং যুক্ত করা)
        $slug = Str::slug($name) . '-' . Str::random(4);

        return [
            'name' => $name,
            'slug' => $slug,
        ];
    }
}

```

**কোডের মূল বিষয়সমূহ:**

* **ফেকার ইনস্ট্যান্স:** `\Faker\Factory::create('bn_BD')` এর মাধ্যমে বাংলাদেশি লোকাল ডাটা জেনারেট করা নিশ্চিত করা হয়েছে।
* **ইউনিক নেম:** ডুপ্লিকেট নেম যাতে না হয় সেই জন্য আমরা `unique()` এবং নেমগুলোকে ওয়ার্ড আকারে আনার জন্য `words(2, true)` ইউজ করছি। এখানে `2` দিয়ে দুটি শব্দ আসবে এবং লাস্টে `true` দিয়ে বোঝানো হচ্ছে এটি স্ট্রিং হিসেবে রিটার্ন করবে।
* **ইউআরএল ফ্রেন্ডলি স্লাগ:** `Illuminate\Support\Str` ক্লাসের হেল্প নিয়ে স্লাগ ফরম্যাট করা হয়েছে। নামের ভেরিয়েবলটি আসার পর ডট কনক্যাটিনেশন হবে, এরপর হাইফেন আসবে এবং লাস্টে ইউনিক বা একটু পার্থক্য করার জন্য রেন্ডমলি ৪ অক্ষরের ইউআরএল ফ্রেন্ডলি কিছু জিনিস জেনারেট হবে।

---

### সারসংক্ষেপ:

ট্যাগ ফ্যাক্টরিতে আমাদের কাজ মূলত ছোট ছোট, খুব বেশি না। এভাবে সেটআপ করে দিলে আমাদের ট্যাগ ফ্যাক্টরির কাজ সম্পন্ন হয়ে যাবে। এরপরে আমরা যেটি করবো সেটি হচ্ছে আমরা পোস্ট ফ্যাক্টরিতে (Post Factory) কাজ শুরু করব। পোস্ট ফ্যাক্টরিটা আমাদের একটু বড় হবে, তাই আমরা এই পর্বটা এখানেই শেষ করছি এবং আগামী পর্ব নতুন একটা ভিডিওতে শুরু করব।
